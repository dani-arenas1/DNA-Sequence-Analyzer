# 🧬 DNA Sequence Analyzer

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![License](https://img.shields.io/badge/license-MIT-green.svg)
![JavaScript](https://img.shields.io/badge/language-JavaScript-f7df1e.svg)
![Bioinformatics](https://img.shields.io/badge/field-Bioinformatics-ff69b4.svg)

Una herramienta web rápida, interactiva y de ejecución 100% local (client-side) para el análisis de secuencias de ADN. Diseñada para bioinformáticos e investigadores que necesitan extraer métricas rápidas de archivos FASTA garantizando la privacidad absoluta de los datos genómicos.

🔗 **[Prueba la aplicación en vivo aquí](https://dani-arenas1.github.io/DNA-Sequence-Analyzer/)**

##  Características Principales

- **Cálculo de Contenido GC:** Métricas instantáneas de la composición de la secuencia.
- **Complementario Inverso:** Generación automática de la hebra antiparalela (`Reverse Complement`).
- **Visualización Gráfica:** Gráficos dinámicos de la composición de bases (A, T, G, C) generados en el navegador.
- **Soporte multi-FASTA:** Carga directa de archivos `.fasta` o introducción manual de secuencias.
- **Exportación de Datos:** Descarga de las métricas generadas en formato CSV para análisis estadístico posterior.
- **Privacidad Total (RGPD):** Al ser una aplicación *Front-end*, ningún dato sale del ordenador del usuario.

## 🚀 Instalación y Uso

No requiere instalación de dependencias, entornos Conda ni servidores backend. Funciona directamente en cualquier navegador moderno.

1. Clona este repositorio en tu equipo local:
   ```bash
   git clone https://github.com/dani-arenas1/DNA-Sequence-Analyzer.git
   ```
2. Entra en la carpeta del proyecto:
   ```bash
   cd DNA-Sequence-Analyzer
   ```
3. Abre el archivo `index.html` haciendo doble clic o arrastrándolo a tu navegador (Chrome, Firefox, Safari).

## 🛠️ Stack Tecnológico

- **Estructura y Estilos:** HTML5, CSS3 moderno (Responsive Design).
- **Lógica Computacional:** Vanilla JavaScript (ES6+).
- **Visualización de Datos:** Funciones nativas de Canvas API / Chart.js.
